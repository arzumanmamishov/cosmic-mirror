# Swiss Ephemeris Data Files

These `.se1` files contain the precise astronomical position data used by the
Swiss Ephemeris library to calculate planet positions, house cusps, and aspects.

## Files
- `sepl_18.se1` — Main planets (Sun, Mercury, Venus, Mars, Jupiter, Saturn, Uranus, Neptune, Pluto)
- `semo_18.se1` — Moon
- `seas_18.se1` — Asteroids (Chiron, Ceres, Pallas, Juno, Vesta)

## Coverage
The `_18` files cover dates **1800-01-01** to **2399-12-31**, which is sufficient
for any user born in modern times.
